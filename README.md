# **Robot Factory**

## **Description**
Robot Factory is a Assembler compiler.
It enables you to transform a .s (Assembly) into a .cor (Machine code).

## **Usage**
To compile the project, use `make re` in a terminal.
Then execute the program using `./asm file[.s]`.
